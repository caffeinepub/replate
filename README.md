# Ashish Singh - Portfolio Website

A professional single-page portfolio website for Ashish Singh, Marketing Innovator & Growth Leader with 18+ years of experience in B2C, B2B, retail digitization, and demand generation.

## Features

- **Responsive Design**: Fully responsive layout optimized for desktop, tablet, and mobile devices
- **Modern Tech Stack**: Built with React 19, TypeScript, Tailwind CSS, and Internet Computer
- **Smooth Navigation**: Fixed header with smooth scrolling to different sections
- **Dark Mode**: Toggle between light and dark themes
- **Professional Design**: Corporate teal and warm neutrals color palette with elegant typography

## Sections

1. **Hero** - Name and tagline with call-to-action buttons
2. **About** - Professional summary and background
3. **Academic Qualifications** - Educational credentials
4. **Skills & Competencies** - Key marketing expertise and achievements
5. **Recognition & Awards** - Notable achievements including Top Gun Award 2024
6. **Newsletter** - Email subscription form
7. **Contact** - Phone, email, and LinkedIn information
8. **Footer** - Useful links and credits

## Design Highlights

- **Typography**: Playfair Display for headings, Source Sans 3 for body text
- **Color Scheme**: Corporate teal primary color with warm amber accents
- **Animations**: Fade-in and slide effects on scroll
- **Achievement Metrics**: Highlighted 425% growth and 66% digitization achievements
